# Method to show the TidySet object

Prints the resulting table of a TidySet object. Does not shown elements
or sets without any relationship (empty sets). To see them use
[`sets()`](https://docs.ropensci.org/BaseSet/dev/reference/sets.md) or
[`elements()`](https://docs.ropensci.org/BaseSet/dev/reference/elements.md).

## Usage

``` r
# S4 method for class 'TidySet'
show(object)
```

## Arguments

- object:

  A TidySet

## Value

A table with the information of the relationships.
